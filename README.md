# Sales-Performance-Analysis (Excel)
## Project Overview
An end-to-end analysis of retail sales and customer data to uncover performance trends and key insights using Excel.The project demonstrates core data analytics skills including data cleaning, exploratory analysis and data visualisation. 
## Dataset
This dataset contains 9800 order-level sales data (Superstore dataset from Kaggle) including customer details, product categories, sales and order dates from 2015-2018.
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
## Data Cleaning Process
The following steps were performed to ensure data accuracy and consistency:
- Converted date columns to proper date formats
- Checked and imputed missing and blank values
- Verified the numeric column (sales) was correctly formatted
- Identified repeated Order IDs and distingished legitimate multi-item orders
- Created derived Order Year and Order Month columns for analysis
 ## Data Dictionary
 A complete data dictionary was created to document all columns in the dataset, including derived fields. This ensures clarity and transparency for anyone reviewing or reusing the data.
 ## Business Questions Answered
 - Which region generates the highest amount of sales?
 - Which product category and sub-category generate the highest amount of sales?
 - How does sales trend over time?
 - How does sales by customer segment trend over time?
## Analysis and Visualisations
Pivot tables and charts were created to support analysis, including:
- Total Yearly Sales by Region
- Total Yearly Sales by Customer Segment
- Total Sales by Product Category
## Key Insights
- There has been a steady growth in sales over the years.
- There has been a fall in total sales in May but a rise in January sales.
- The technology category has contributed the mist to sales over the last four years.
